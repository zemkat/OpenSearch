Press the button below to install the MARC Geographic Areas search plugin.

This plugin accepts a Geographic Area name (such as Spain or Kentucky) and includes a code suitable for inclusion in MARC bibliographic field 043.

Press the button below to install the MARC Countries search plugin.

This plugin accepts a Country/State/Province (such as Spain or Kentucky) and includes a code suitable for inclusion in the MARC fixed field Ctry.

Press the button below to install the MARC languages search plugin.

This plugin accepts a Language (such as Spanish) and includes a code suitable for inclusion in the MARC fixed field Lang or the MARC field 041.
